# eyeonwater_for_ha
EyeOnWater integration for Home Assistant by Dave Strickler
- This app runs once an hour and fetches the latest data from eyeonwater.
- When configuring it, you'll need your username and password to the [EyeOnWater](https://eyeonwater.com/) or their mobile app.
- When you first install this app, it might not show water usage under "Energy" in the dashboard. Wait an hour and it should update with real usage.
- It's common for a tiny amount of water to be used when you're not in the house, such as 0.06 gallons per day.
