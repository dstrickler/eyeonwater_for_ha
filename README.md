# eyeonwater_for_ha
EyeOnWater integration for Home Assistant
- This app runs once an hour and fetches the latest data from eyeonwater.
- When you first install the app, it might not show water usage under "Energy" in the dashboard. Wait an hour and it should update with real usage.
- It's common for a tiny amount of water to be used when you're not in the house, such as 0.06 gallons per day.
